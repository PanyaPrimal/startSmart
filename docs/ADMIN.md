# Керівництво редактора — адмінка StartSmart

Цей документ — для людини, яка редагує контент сайту (тексти, ціни, фото, викладачів). Програмувати не треба.

## Що таке адмінка

Адмінка — це графічний редактор на базі **Sveltia CMS**, який живе за адресою `/admin` вашого сайту (наприклад, `https://startsmart.com/admin/`). Вона зберігає зміни прямо в файли проєкту — після збереження зміни з'являються на сайті автоматично (через кілька секунд у dev-режимі або після деплою у проді).

## Як зайти

### Локально (на своєму комп'ютері, для перевірки перед публікацією)

Треба один раз встановити Node.js ([nodejs.org](https://nodejs.org)), далі в терміналі:

```bash
# у першому вікні терміналу — запускаємо сайт
npm install      # тільки перший раз
npm run dev

# у другому вікні — запускаємо proxy для CMS
npx @sveltia/cms-proxy-server
```

Відкрий у браузері: **http://localhost:4321/admin/**

Авторизація не потрібна — це локальна адмінка. Усі зміни зберігаються прямо у файли на диску.

### На продакшені

Після налаштування GitHub OAuth (див. `docs/DEPLOY.md`) адмінка буде доступна за адресою `https://startsmart.com/admin/` — увійти можна через GitHub-акаунт, який має доступ до репозиторію.

## Що можна редагувати

У лівій панелі адмінки є 4 розділи:

### 1. Налаштування сайту (Settings)

Один файл — глобальні речі, які показуються на всіх сторінках:
- **Назва** сайту та слоган
- **Email** для контактів і футера
- **Соцмережі**: Facebook, Telegram, Instagram URL
- **Меню** (Nav) — пункти верхнього меню. Кожен = напис + посилання
- **Footer copyright** — рядок унизу сторінки

### 2. Викладачі (Teachers)

Кожен викладач — окремий запис. Кнопка «New» створює нового.

Поля:
- **Ім'я** — як відображається на сайті (великими літерами)
- **Зображення** → `Файл` (головне фото), опційно `Файл (темна тема)` (якщо хочеш іншу версію для темної теми), `Alt` (опис для доступності)
- **Цитата** — фраза в лапках під іменем
- **Інформація** — один довгий рядок з роздільниками `·` (рівень · досвід · освіта · …)
- **Мови** — галочки: english/polish/ukrainian/russian
- **Порядок** — число, за яким сортувати (менше = раніше)

Щоб викладач показався на сторінці English School або Szkoła Językowa, треба в **Сторінках** (див. нижче) додати його в масив «Викладачі».

### 3. Формати навчання (Formats)

Три формати: груповий, парний, індивідуальний. Можна додавати нові.

Поля:
- **Повна назва** — для заголовка на сторінці формату (напр. «ГРУПОВІ ЗАНЯТТЯ»)
- **Коротка назва** — для карточок на сторінці English School (напр. «В групі»)
- **Підзаголовок** — опис під заголовком hero
- **Ціна від** — короткий рядок (напр. «від 150 грн / 80 хв.»), показується в карточках
- **Hero зображення** — фонова картинка hero-секції
- **Hero оверлей** — колір плашки поверх картинки: `pink`, `teal` або `dark`
- **Іконка для картки** — маленьке фото-іконка в сітці «Види навчання»
- **Опис у картці** — 1-2 речення для цієї ж сітки
- **URL сторінки** — куди веде посилання (напр. `/group_classes`)
- **Примітка до цін** — підпис над тарифами (напр. «Ціна на навчання з викладачем Світлана»)
- **Тарифи** — список. Кожен тариф:
  - Назва (напр. «ПЕРШИЙ УРОК»)
  - Тривалість
  - Ціна
  - Підпис (за урок / пакет / …)
  - Посилання (де має відкритися після кліку на «Записатися»)
  - Напис кнопки
  - Виділений? — галочка, якщо це головний тариф (отримає рожеву рамку)
- **FAQ** — список питань-відповідей для цього формату
- **Порядок** — для сортування

### 4. Сторінки (Pages)

Три сторінки: `home` (головна з кнопками), `english_school`, `polish`. Це самі насичені записи.

Загальні поля:
- **Заголовок (SEO)** — показується у вкладці браузера
- **Hero-секція** — великий банер угорі (заголовок, підзаголовок, картинка, оверлей, CTA кнопка)
- **Секція "Про"** — label, заголовок, абзаци, галочки-список, картинка
- **Секція Features** — заголовок + список іконок/написів
- **Методи** — посилання на формати (Individual/Pair/Group). Додається через віджет relation — просто вибери зі списку.
- **Тарифи на сторінці** — короткі картки цін на головну сторінку школи
- **Викладачі** — посилання на записи з розділу Викладачі
- **FAQ** — список питань-відповідей
- **Секція контактів** — заголовок, текст, заголовок форми
- **Quick-menu** — тільки для `home`: заголовок + кнопки-посилання

## Як зберегти зміни

1. Внеси зміни у полях.
2. Натисни **Save** (вгорі праворуч).
3. На продакшені: Sveltia створить коміт у GitHub від твого імені. За 1-3 хвилини сайт пересобереться автоматично (якщо налаштовано CI).
4. Локально: зміни одразу у файлах. Щоб опублікувати — зроби `git push` з терміналу (або попроси розробника).

## Як додати нове зображення

У будь-якому полі типу `image` — натисни на нього → «Upload» → вибери файл з комп'ютера. Файл завантажиться у `public/images/` і автоматично підставиться.

**Порада:** стискай фото перед завантаженням ([tinypng.com](https://tinypng.com)), інакше сайт буде довго вантажитися.

## Як додати ще одного викладача

1. Розділ **Викладачі** → **New Викладач**.
2. Заповни всі поля.
3. Збережи.
4. Відкрий потрібну сторінку (напр. English School) → знайди поле **Викладачі** → **Add** → вибери зі списку щойно створеного викладача.
5. Збережи сторінку.

## Як додати новий формат навчання

Сам формат створити просто (розділ Формати → New), але щоб з'явилася окрема сторінка (як `/group_classes`), треба щоб розробник додав файл `src/pages/<slug>_classes.astro`. Просто напиши йому slug нового формату.

## Часті питання

**Я зіпсував сторінку, як відкотити?**
Усі зміни зберігаються в git. Попроси розробника повернути попередню версію — це 1 команда.

**Чи побачать редагування відвідувачі одразу?**
Ні, має пройти деплой (на Cloudflare Pages — 1-2 хвилини після save).

**Де знайти файли, якщо хочу редагувати руками?**
`src/content/settings/site.json`, `src/content/teachers/*.json`, `src/content/formats/*.json`, `src/content/pages/*.json`. Формат — звичайний JSON.

**Темна тема — як задати окреме фото?**
У будь-якому полі `Зображення` є опційний `Файл (темна тема)` — якщо його не заповнити, у темній темі використається основне фото з автоматичним приглушенням (brightness 85%).
